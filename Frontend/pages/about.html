About FormatForge

Project Overview

FormatForge is a smart Image-to-PDF converter developed as a final-year project. It allows users to convert images into high-quality PDF files with optional image enhancement for better clarity.

Key Features

  • Image to PDF conversion
  • Smart image enhancement (brightness & contrast)
  • Clean and responsive UI
  • Secure processing (no data stored permanently)
  • Fast and lightweight backend using FastAPI

Technologies Used

  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Backend: Python (FastAPI)
  • Database: SQLite (for history – optional)
  • Tools: VS Code, Git

Project Objective

The objective of FormatForge is to provide a simple yet powerful document conversion tool that goes beyond basic converters by offering smart image enhancement and a modern SaaS-style UI.

Developed as part of an academic final-year project — 2025
